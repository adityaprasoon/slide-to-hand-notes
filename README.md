**How to use**

- Download the repository
- Have Python 3 installed on your machine
- From inside the repository directory, run `python slideToNodes.py /path/to/slideFile.pdf`
- Find the two new files generated at the original file's location named - `xyz-print-odd.pdf` and `xyz-print-even.pdf`
- Print the odd file first
- Insert the printed pages back in the printer, this time to print on the back side
- Print the even file
- Your slides are now printed for you to take notes on
